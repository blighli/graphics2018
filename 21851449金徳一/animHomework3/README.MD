金砸 21851449  
-----------------------------------
复杂三维模型显示，提交时间：2019年1月8日  
1、支持以下三维模型文件格式中至少1种：  
OBJ  
3DS  
STL  
2、支持多个光源的光照效果  
3、支持多种视点浏览方式  
以模型为中心的平移旋转和缩放  
以视点为中心的场景漫游  
-----------------------------------
完成情况：  
1.可打开.OBJ模型文件。  
2.一左一右两个光源，各方面参数有较大区别，使得茶壶表现出较好的综合光照效果。  
3.以模型为中心的平移（鼠标右键）、旋转（鼠标左键）、缩放（滚轮或键盘‘W’‘S’）；  
以视点为中心的场景漫游：左右旋转（键盘←→）、前进后退（键盘↑↓）
